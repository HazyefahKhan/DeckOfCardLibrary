# DeckOfCardLibrary
An implementation of a deck of playing cards containing 52 cards in thirteen ranks (two through Ace) and four suits (Clubs, Diamonds, Hearts, and Spades).

### How to build and use this library:

  1. From the project folder run: javac -d ./build *.java
  2. Once it creates the build directory, within the build directory: jar cvf YourJar.jar *
  3. You can add the jar file to your build path's external archive to use the library.
